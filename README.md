# homebrew-tap

Homebrew tap for [Eyebreak](https://github.com/Mirxomitov/eyebreak) — a macOS
20-20-20 eye-break timer.

## Install

Native menu-bar app (recommended):

```sh
brew install mirxomitov/tap/eyebreak
brew services start eyebreak
```

SwiftBar plugin version:

```sh
brew install --cask swiftbar
brew install mirxomitov/tap/eyebreak-swiftbar
eyebreak-swiftbar
```
